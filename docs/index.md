# What is Pyra Chain?

## Overview

**Pyra Chain** is a petition-focused blockchain with optional privacy, advancing validation mechanisms through ecosystem integration. Built using Solana's open-source code, Pyra Chain ($PYRA) delivers high-performance infrastructure with **41K TPS on a single validator node** while providing a complete ecosystem for developers, validators, and communities.

Our network features a comprehensive vault system for sustainable tokenomics, including treasuries for marketing, development, community rewards, validators, grants, and more. This structure ensures long-term growth and fair distribution without team-controlled supply manipulation.

## Why Choose Pyra Chain?

Pyra Chain combines the proven performance of Solana's architecture with advanced validation mechanisms and optional privacy features. Our petition-focused approach enables transparent governance while the comprehensive vault system ensures sustainable development.

With deep liquidity, audited smart contracts, and community-first tokenomics, Pyra Chain provides the infrastructure for building the next generation of decentralized applications.
