# Create Token

## Launch Your Token in Seconds

**Create your own utility or community token** on Pyra Chain with **instant deployment** - no coding required. Every token gets **instant DEX listing**, **explorer visibility**, and **bonding curve trading** out of the box.

- **Cost**: **1,000 $PYRA** (burned on creation → deflationary)
- **Network**: Pyra Chain (bridge $PYRA first if needed)
- **Time**: < 30 seconds

---

## Step-by-Step Guide

### Step 1: Access Token Creator
Go to the official launchpad and connect your wallet.

**Action**:
1. Visit [pyrachain.io/launch](https://pyrachain.io/launch)
2. Click **"Connect Wallet"** (top-right)
3. Click **"Create Token"**

---

### Step 2: Fill Token Details
Enter your token information. **Live Preview** updates in real time.

| Field | Requirement | Example |
|------|-------------|--------|
| **Token Name** | Required | `MyToken` |
| **Token Symbol** | Required, uppercase | `MTKN` |
| **Description** | Required | `Community token on Pyra Chain` |
| **Token Image** | Required | Upload PNG/JPG (≤5MB) |
| **Website** | Optional | `https://mytoken.io` |
| **Telegram** | Optional | `@mytoken` |
| **Twitter/X** | Optional | `@mytoken` |

**Live Preview** shows:
- Price, market cap, 24h volume, holders
- Bonding curve progress
- Buy/Sell buttons

---

### Step 3: Upload Image
Add a high-quality logo for your token.

**Action**:
1. Click **"Upload token image"**
2. Select **PNG or JPG** file (≤5MB)
3. Recommended: **512x512px**, transparent background
4. Preview appears instantly

> **Tip**: Use a bold, simple design - it shows on DEX, explorer, and wallets.

---

### Step 4: Create Token
Pay the creation fee and deploy.

**Action**:
1. Confirm all fields
2. Click **"Create Token (1K $PYRA)"** (purple-pink gradient button)
3. Approve transaction in your Solana wallet:
   - **Fee**: 1,000 $PYRA (burned)
   - **Gas**: ~0.0001 $PYRA
4. Wait ~10 seconds → **Token Created!**

---

## After Creation
